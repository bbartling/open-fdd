# Using the React dashboard

The **React dashboard** is not part of the PyPI **`open-fdd`** package or this engine repository. It lives in **[open-fdd-afdd-stack](https://github.com/bbartling/open-fdd-afdd-stack)** under [`frontend/`](https://github.com/bbartling/open-fdd-afdd-stack/tree/main/frontend) (Vite, TypeScript, tests).

- **Operator docs:** [Using the React dashboard](https://bbartling.github.io/open-fdd-afdd-stack/frontend) on the stack documentation site  
- **UI evaluation notes:** [React frontend evaluation](https://bbartling.github.io/open-fdd-afdd-stack/frontend_evaluation) (stack site)

For Caddy, ports, and JWT login, see the stack’s [Security](https://bbartling.github.io/open-fdd-afdd-stack/security) page.
