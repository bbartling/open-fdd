# API reference

| API | Audience | Doc |
|-----|----------|-----|
| **Operator Bridge** (REST) | Integrators, dashboard, deployment AI | [Bridge API](bridge_api) |
| **`open_fdd.engine`** | Offline YAML rules on pandas | [Engine API](../api/engine) |
| **`open_fdd.reports`** | Fault summaries and plots | [Reports API](../api/reports) |
| **`open_fdd.schema`** | Result/event models | Used by engine; see engine doc |

Production buildings use the **bridge** + Rule Lab Python. PyPI engine is for CSV/notebook and CI regression.
