# Open-FDD Documentation

## 📌 Overview

`open-fdd` is an **open-source Fault Detection and Diagnostics (FDD) tool** designed for analysts and engineers working with HVAC data using **Pandas** in Jupyter Notebooks or local analysis workflows. This library is not necessarily an IoT tool for large-scale cloud deployments but rather a **flexible, lightweight** framework for detecting HVAC system anomalies using **ASHRAE** and **NIST-inspired fault equations**.
