# Open-FDD 3.2.2 Release Notes

**Merged to `master`:** 2026-06-27  
**GHCR image:** `ghcr.io/bbartling/openfdd-edge-rust:3.2.2` (also `:latest` on master push)

## Highlights

- **Live OT drivers** — BACnet and Modbus require `OPENFDD_*_MODE=live`; simulated OT paths removed
- **BACnet** — RPM discovery/poll, priority-array override scan, CSV retention, local Open-FDD BACnet server
- **Wiresheet UI** — FDD Studio (`/wiresheet`), CSV Fusion wiresheet (`/csv`), Model hub
- **Haystack / SQL UX** — cleaner integration pages, larger SQL editor
- **Guards** — no hardcoded bench IDs, `no_hardcode_audit` tests

## Upgrade

```bash
cd ~/open-fdd   # or your install path
git checkout master && git pull
./scripts/openfdd_rust_site_update.sh
curl -s http://127.0.0.1:8080/api/health | jq '{version, image_tag}'
```

Never run `docker compose down -v`.
