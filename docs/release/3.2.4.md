# Open-FDD 3.2.4 Release Notes

**Branch:** `master` (merged via PR #415, #416, #419)  
**GHCR images:** `ghcr.io/bbartling/openfdd-edge-rust:3.2.4`, `ghcr.io/bbartling/openfdd-mcp:3.2.4`

## Highlights

- **Agent-first CSV ingest:** fail-closed preflight gate, `GET /api/ingest/contract`, MCP playbook tools
- **SQL FDD workbench:** Grafana-style DataFusion query UI
- **Building insight:** 15-minute cached HVAC summary with optional Ollama
- **Bench validation:** strict driver gate, `docs/bench-validation.md`, safe restart helper
- **Haystack RDF:** OxiGraph semantic model cleanup (PR #416)
- **Rust release workflow:** manual `rust-release.yml` → GHCR + GitHub Release (Python-era packages archived)

## Upgrade

```bash
cd ~/open-fdd
./scripts/openfdd_rust_site_backup.sh
NEW_TAG=3.2.4 ./scripts/openfdd_rust_site_update.sh
./scripts/openfdd_rust_edge_validate.sh
curl -s http://127.0.0.1:8080/api/health | jq '{version, image_tag}'
```

## Publish (maintainers)

```bash
gh workflow run rust-release.yml -f version=3.2.4 -f prerelease=false
```
