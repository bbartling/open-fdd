Data model

Sites → Equipment → Points (from the database). Left-click a row with ▶ to expand; right-click to delete. brick_model.ttl is generated from this tree. To show BACnet devices (e.g. BensFakeAhu) here, create them as Equipment under a site via API or future import; points under equipment become time-series refs. Data viewing is in Grafana.

    BACnet driver

    The Open-FDD backend (API container) calls this URL. In Docker, localhost inside the container is not the host—use the prefilled URL from OFDD_BACNET_SERVER_URL (e.g. http://host.docker.internal:8080 when bacnet-server runs on the host). For remote buildings, set the URL to that BACnet gateway.

    Example: http://localhost:8080 (same host) or http://192.168.1.10:8080 (OT LAN device).
