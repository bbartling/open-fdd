# openfdd-engine (deprecated)

**Do not publish this package on new releases.** Use PyPI **`open-fdd`** only:

```bash
pip install open-fdd
```

```python
from open_fdd.engine import RuleRunner
from open_fdd.playground import compile_evaluate, sweep_rule
```

This directory remains for legacy **`import openfdd_engine`** editable installs only. CI no longer builds or publishes **`openfdd-engine`** to PyPI.

See [PyPI releases (`open-fdd`)](../../docs/howto/openfdd_engine_pypi.md).
